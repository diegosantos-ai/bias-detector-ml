# Changelog

## [v0.1.0] - 2026-02-08

### Adicionado
- **Detecção de Viés**: Implementação inicial do modelo usando `sentence-transformers` e `scikit-learn` para detectar viés de gênero, idade e cultura.
- **API REST**: Endpoint `/analyze` criado com FastAPI.
- **MLflow Tracking**: Rastreamento de experimentos e registro de modelos.
- **Gerador de Dados**: Script para criação de datasets sintéticos de treinamento.
- **Documentação**: README traduzido para Português, Guia de Contribuição e Código de Conduta.
