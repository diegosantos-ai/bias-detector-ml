# Guia de Contribuição

Obrigado por considerar contribuir para o **Detector de Viés em RH**!

## Como Começar

1. **Fork** este repositório.
2. Clone seu fork:
   ```bash
   git clone https://github.com/diegosantos-ai/bias-detector-ml.git
   ```
3. Crie uma branch para sua feature (`git checkout -b feature/nova-feature`) ou correção (`git checkout -b fix/bug-fix`).

## Ambiente de Desenvolvimento

Certifique-se de ter Python 3.10+ instalado.

```bash
# Criar ambiente virtual
python -m venv .venv

# Ativar (Windows)
.venv\Scripts\activate

# Instalar dependências
pip install -r requirements.txt
```

## Padrões de Código

- **Idioma**: Toda a documentação, comentários e mensagens de commit devem ser em **Português**.
- **Testes**: Adicione testes unitários para novas funcionalidades em `tests/`. Execute `pytest` antes de enviar.
- **Estilo**: Mantenha o código limpo e siga as convenções do projeto.

## Pull Requests

1. Certifique-se de que todos os testes passaram.
2. Abra um Pull Request descrevendo suas mudanças.
3. Aguarde a revisão.
